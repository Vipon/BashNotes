# List of command

All command execute in the next order. You should press Ctrl-b, release Ctrl-b, and only after it press the next letter.

* New window
	* Ctrl-b c
* List of windows
	* Ctrl-b w
* Switch windows
	* Ctrl-b n (next)
	* Ctrl-b p (prev)
	* Ctrl-b [NUMBER]
* Split window
	* Ctrl-b % (by colon)
	* Ctrl-b " (horizontal)
* Close window
	* Ctrl-b x
* Quit session
	* Ctrl-b d
